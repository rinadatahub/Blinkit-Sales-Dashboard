## Blinkit-Sales-Dashboard(Interactive-Sales-Dashboard creation using Blinkit)
Power BI dashboard analyzing Blinkit sales data with insights on product performance, sales trends, and customer behavior.

## Project Objective 
1) Tracking business performance
2) Identifying top-selling products
3) Understanding customer demand
4) Supporting data-driven decision-making

## Dataset Used 
-<a href="https://docs.google.com/spreadsheets/d/1wB2ED1tMgRBCyRKdpdxfD3isQ8YsMLI5/edit?usp=drive_link&ouid=113744491659665008374&rtpof=true&sd=true">Dashbord</a>

## Question (KPI's)
1) What is the total sales revenue?
2) What is the total number of orders?
3) Which product category generates the highest sales?
4) What are the top-selling products?
5) Which outlet/location has the highest sales?
6) What is the average purchase value per customer?
7) Where can the business improve sales?

## process 
Data Collection
Collected Blinkit sales dataset containing information about products, orders, outlets, and sales.

1. Data Cleaning
   Cleaned the dataset using Power Query by handling missing values, removing duplicates, and correcting data types.
2. Data Transformation
   Transformed raw data into a structured format and created necessary columns for better analysis.
3) Data Modeling
   Built relationships between tables and designed a data model to ensure accurate analysis.
4) DAX Calculations
   Created measures such as Total Sales, Total Orders, and Average Order Value (AOV) using DAX.
5) Data Visualization
   Designed interactive visuals like:
6) Bar charts
a. Line charts
b. Pie/Donut charts
c. KPI cards
7) Dashboard Development
   Developed an interactive dashboard with slicers and filters to allow dynamic data exploration.
8) Insight Generation
   Analyzed the visuals to identify trends, top-performing products, and business opportunities.

## Project Insigths 
Top categories generate the highest sales, showing strong customer demand in specific product segments.
1. A few products contribute most of the revenue, indicating a high-performing product group.
   Certain outlets outperform others, suggesting location and demand differences impact sales.
2. Sales show clear trends over time, with peak periods where customer activity is highest.
3. Average order value is stable, but can be improved by increasing basket size.
4. Customer buying patterns show repeated purchases, indicating loyal customers.
5. Some categories/products have low performance, highlighting areas for improvemen

## Final Conclusion 
The Blinkit Dashboard shows that overall sales performance is mainly driven by a few top-performing products and categories. Certain outlets generate higher revenue, indicating that location and customer demand play a key role in business success.
The analysis also highlights consistent customer purchasing patterns and stable average order value, while identifying low-performing products that need improvement. Time-based trends reveal peak sales periods, which can be used for better planning and promotions.
Overall, this project demonstrates how data visualization in Power BI can help businesses make data-driven decisions, improve sales strategies, and optimize product performance.
